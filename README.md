# Containers
Реализация базовых контейнеров

Данный проект явялется командным. Он был выполнен двумя людьми. Моя часть работы - реализация красно-черного дерева (основа map, set), а также следующих контейнеров:
- list
- map
- set
- vector

Весь список:
* array
* list (двусвязный список)
* map (красно-черное дерево)
* set (красно-черное дерево)
* multiset
* queue
* stack
* vector

У всех контейнеров есть базовый набор методов, представленных в стандартной библиотеке.
