# Containers
Реализация базовых контейнеров

Данный проект явялется командным. Он был выполнен двумя людьми. Моя часть работы - реализация красно-черного дерева (основа map, set), а также следующие контейнеры:
- list
- map
- set
- vector

Весь список реализованных контейнеров:
* array
* list (двусвязный список)
* map (красно-черное дерево)
* set (красно-черное дерево)
* multiset
* queue
* stack
* vector

У всех контейнеров была реализована базовая составляющих контейнеров, представленных в стандартной библиотеке.
